# 固件的编译和下载
Windows用户可以在GitBash中执行make bin来实现固件的编译。需要提前按照SDCC编译工具。

短接芯片的DP引脚和VCC引脚，然后上电，芯片将进入Boot模式。

使用WCHISPTool工具上传固件。
